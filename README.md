# README

This is a Big class assignments ,  the goal is to realize handwritten digit recognition.

The Operating Environment is  OpenCV 3.4.13 + ubuntu20.04 + cmake(Version 3.1 at least) 

If you want to copy this code, pay attention to your file paths in addition to your environment. All paths in this code are absolute paths.



## using

put  your ImagePath in the use/use.cpp    `string Imgpath = "your ImagePath";` , then 

`cd use`

`mkdir build`

`cd build`

`cmake ..`

`make`

`./use`
